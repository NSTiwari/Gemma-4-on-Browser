# Gemma 4 on Browser
This repository is an implementation of on-device inferencing Gemma 4-E2B-it on the browser for image, audio, and text inputs.

## Steps to install/run:

1. Clone the repository on your local machine.
2. Navigate to `Gemma-4-on-Browser` directory.
3. Run `npm install` to install the packages.
4. Run `node server.mjs` to start the server.
5. Open `localhost:3000` on your web browser and start inferencing with Gemma 4.

> [!NOTE]
> Depending upon your internet speed, it will take several minutes to download the model weights for the first time.
